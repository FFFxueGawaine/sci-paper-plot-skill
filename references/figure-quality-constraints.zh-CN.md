# SCI 论文图片基本约束

这个清单用于指导当前 skill 生成、复刻和检查论文图片。核心原则是：**布局位置可配置，语言字体可选择，但同一篇论文必须统一**。

## 1. 语言和字体

| 元素 | 默认建议 | 可选情况 |
|---|---|---|
| 坐标轴标签 | 英文，Times New Roman | 中文论文或中文报告可用中文 |
| 图例 | 英文，Times New Roman | 中文报告可用中文 |
| 图标题 | 英文，Times New Roman | 中文标题可用宋体 `SimSun` |
| 数学公式 | Matplotlib mathtext/STIX | 不建议混用多套数学字体 |
| 单位 | 英文单位符号，空格后括号 | 例如 `time (s)`、`频率 (Hz)` |

标题字体不是固定规则。英文 SCI 图通常用 Times New Roman；中文报告或中文图题可以用宋体。模板中可写：

```python
from scimplstyle_mssp import set_panel_title

set_panel_title(ax, "误差分布", lang="zh")
set_panel_title(ax, "Error distribution", lang="en")
```

如果系统没有宋体，Matplotlib 可能回退到其他字体。正式导出前需要检查 PNG/PDF 中中文是否正常显示。

## 2. 子图标号 `(a)(b)(c)` 的位置

子图标号不能固定为唯一位置。应根据期刊模板、图形密度和版面选择：

| 位置 | 适合情况 | 写法 |
|---|---|---|
| 上方居中 | 多数英文 SCI 图、子图上方留白充足 | `panel_label(ax, "(a)")` 或 `panel_label(ax, "(a)", loc="top-center")` |
| 左上角内部 | 图上方空间紧、期刊偏好左上角标号 | `panel_label(ax, "(a)", loc="upper-left")` |
| 下方居中 | 期刊要求子图标号在图下方，或每个子图下方有独立说明 | `panel_label(ax, "(a)", loc="bottom-center")` |
| 左下方 | 横向排版较密，下方仍需左对齐 | `panel_label(ax, "(a)", loc="lower-left")` |

选择原则：

- 同一张多子图内的位置必须统一。
- 同一篇论文内最好统一，除非某类图有明确期刊要求。
- 标号不能压住曲线、散点、色条或局部放大框。
- 下方标号会增加底部留白，保存时需要确认 x 轴标签没有被裁切。

## 3. 布局约束

- 单栏图宽约 `85 mm`，双栏图宽约 `170-180 mm`。
- 多子图优先使用规则网格：`1 x 3`、`2 x 2`、`2 x 3`。
- `constrained_layout=True` 优先，保存时保留 `pad_inches=0.04` 或更大。
- 色条、图例、局部放大框不应挤压主要数据。
- 下方标号、长 x 轴标签、中文标题同时出现时，需要增大图高或底部留白。

## 4. 标签和单位

- 单位前留空格：`time (s)`，不要写 `time(s)`。
- 英文 SCI 图优先短标签：`dis. (mm)`、`vel. (mm/s)`、`fre. (Hz)`。
- 中文报告可写：`位移 (mm)`、`频率 (Hz)`、`误差 (mm)`。
- 同一篇论文里同一物理量只用一种写法，不要混用 `frequency`、`fre.`、`频率`。

## 5. 导出检查

导出后至少检查：

- 图片不是空白。
- 子图标号位置一致。
- 中文标题或宋体标题正常显示。
- 坐标轴标签、图例、色条没有被裁切。
- 图例没有遮挡关键数据。
- PNG 为 `600 dpi`；需要矢量时再导出 PDF/SVG。

## 6. 推荐默认值

- 英文 SCI 投稿：英文标题和标签，Times New Roman，`panel_label(..., loc="top-center")` 或期刊指定位置。
- 中文报告：中文标题可用宋体，坐标轴可中文或英文，但同一文档统一。
- 期刊模板要求子图标号在下方时：使用 `loc="bottom-center"`，并检查底部裁切。
